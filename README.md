# The Kwarto Product Landing Page
A responsive single-page product landing page for The Kwarto, a cozy and modern condo stay in Baguio City

## Features
- **Navigation Bar**
    - Smooth scrolling to different sections
    - Responsive mobile menu toggle

- **Hero Section**
    - Eye-catching headline
    - Background Image (kwarto unit)
    - Call-to-action "Book Now" button

- **Features Section**
    - Highlights key benefits:
    - Cozy & relaxing space
    - City view balcony
    - Free Wifi
    - Fully equipped kitchen

- **Gallery Section**
    - Responsive image grid
    - Hover zoom effect

- **Details Section**
    - Description of the condo
    - Price per night
    - Amenities list
    - "Reserve Now" button

- **Responsive Design**
    - Works on desktop and mobile devices

- **Interactive Elemetns**
    - Mobile menu toggle
    - Smooth scrolling navigation
    - Booking button alert interaction

## Technologies Used
- **HTML5**-Semantic Structure
- **CSS3** - Flexbox, Grid, Media Queries
- **JavaScript** - Interactivity and DOM manipulation
- **Git & Github** - Version Control
